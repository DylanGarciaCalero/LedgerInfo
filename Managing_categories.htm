Managing categories

To organize your expenses, Ledger organizes them into different categories, such as groceries and transportation. There is no limit on how many categories you can add to group your expenses properly. By sorting your recent expenses accordingly, it is easier to calculate your total spending and view it by category.

To add a new expense category, follow these steps:

  1. Select Add new category from the category section on the homepage.

    Note: You will be redirected to the New Category page.

  2. Enter the category name into the New Category section.
  3. Choose the amount to allocate for the new category.
  4. Select Add category.
  5. Select Add another category to create more categories.

You successfully added a new category and can now start organizing your entries.

Follow these steps to delete a category:

  1. Select Delete a category under the New category section.
  2. Choose which category you want to remove.
  3. Select Ok to confirm the action.

You successfully deleted a category.

To edit a category, follow these steps:

  1. Select Edit a category under the New category section.
  2. Choose which category you want to edit.
  3. Select Ok to confirm the action.

You successfully edited a category.
