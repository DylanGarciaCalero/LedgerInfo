How to understand Ledger's homepage

The homepage contains five sections that display a summary of essential information. You can see your available budget and recent spending from the homepage, and you can also add an expense entry or a new category. The interface layout can change depending on the platform you are using to access your Ledger’s account.


Please refer to "Table 3: Ledger App Icon Legend" for more information.

 

Ledger App Icon Legend for Homepage View.

 

Letter Section Description
A Main menu

Contains links to Ledger's expenses and savings features. From the main menu you can go to different pages of the application.  
B Available budget Displays the available budget you have left.
C Categories Provides an overview of the amount allocated for a category and an option to create a new category
D Recent spendings Provides a summary of your recent spending and access to your expenses.
E Add expense Allows you to add new expenses. You can enter the name, amount, and choose which category you want to add your new transaction.
